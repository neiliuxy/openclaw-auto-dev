# Issue #27 需求规格说明书

## 1. 概述
- **Issue**: #27
- **标题**: feat: 添加 C++ 文件搜索工具 file_finder
- **处理时间**: 2026-03-21

## 2. 需求分析

### 原始描述
需要一个快速的文件搜索工具，可以根据文件名模式在目录树中查找文件。

### 功能点拆解

| ID | 功能点 | 描述 | 验收标准 |
|----|--------|------|----------|
| F01 | 递归搜索目录树 | 遍历指定目录的所有子目录 | 搜索结果包含深层文件 |
| F02 | --name 按文件名模式搜索 | 支持通配符 * 和 ? | --name "*.cpp" 能找到所有 .cpp 文件 |
| F03 | --type 按类型搜索 | f=文件，d=目录 | --type d 只返回目录 |
| F04 | --max-depth 限制深度 | 限制递归层数 | --max-depth 1 只搜当前目录 |
| F05 | --exclude 排除目录 | 排除指定目录 | --exclude ".git" 不搜索 .git |
| F06 | 输出文件路径和大小 | 格式: path (size) | 输出格式正确 |
| F07 | 编译通过无警告 | C++17 标准编译 | g++ -std=c++17 -fsyntax-only 通过 |

## 3. 技术方案

### 3.1 文件位置
`src/file_finder.cpp`（新建）

### 3.2 核心模块
- `main()` — 命令行解析 + 搜索调度
- `match_pattern()` — glob 模式转正则匹配
- `format_size()` — 字节大小格式化

### 3.3 依赖
- C++ 标准库（filesystem, regex, chrono）
- C++17 标准
