# .pipeline-state/ — Pipeline Runtime State

> **⚠️ Runtime state directory. Do not commit stage files.**

This directory contains runtime pipeline state files generated by the OpenClaw Pipeline system.

## File Formats

### Legacy Format (deprecated)
```
<issue_number>_stage   # Plain text integer (0-4)
```

### JSON Format (current)
```
<issue_number>_stage   # JSON: {"issue": N, "stage": N, "updated_at": "...", "error": null}
stage_<N>_<agent>.json # Agent-specific state files
```

## Files

| File | Description |
|------|-------------|
| `<issue>_stage` | Pipeline stage state for a specific issue (0=NotStarted, 1=ArchitectDone, 2=DeveloperDone, 3=TesterDone, 4=PipelineDone) |
| `stage_<N>_<agent>.json` | Agent-specific state (Architect, Developer, Tester, etc.) |

## What NOT to Commit

- `*_stage` files (runtime state)
- `stage_*.json` files (runtime state)
